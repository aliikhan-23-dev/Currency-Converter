💱 Currency Converter Web App
A simple and responsive currency converter built with HTML, CSS, and JavaScript. This project uses the Frankfurter API to fetch real-time exchange rates between multiple currencies.

🚀 Features
Convert between 25+ major world currencies

Real-time exchange rate fetching

Auto-populated currency dropdowns with country flags

Responsive and minimal user interface

🛠 Tech Stack
HTML

CSS

JavaScript (Vanilla)

Frankfurter API

Flags API for country flags
